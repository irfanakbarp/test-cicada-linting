## Getting started

```
// Generate APP_KEY
go run . artisan key:generate

// Install dependencies
go mod tidy

// Run app without hot reload
go run . 

// Run app with hot reload
air

// Go Artisan, use go artisan to create models, requests, middlewares, etc.
go run . artisan --help

```
